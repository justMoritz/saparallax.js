SA(scrollimate) Prallax Sample Page

Works on absolutely and relatively positioned elements!
(this element has a data-bgscroll attribute of 1.5)

About



SA Parallax works simply by inluding jQuery, the saparallax.js file, and the data-sabglayer attribute to any dom element.

By giving a value to the data-sabglayer attribute, you can control the scroll speed of the elements. Enter a number from 0 (same as foreground) to 2 (will appear static).


Sample Application

(this element has a no data-bgscroll attribute set, therefore defaults to 1)

Above is a in image in a figure element. the image is roughly twice the height as the figure, the figure is styled with overflow:hidden. Applied to the image within the figure is the data-sabglayer attribute, which causes each element with that style applied to scroll slower than the foreground.

Of course you can apply the data-bgscroll attribute to any Dom element, just be careful, you might get results that look kind of weird, like this
